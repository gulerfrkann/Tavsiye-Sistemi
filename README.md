# Tavsiye Sistemleri Projesi

Bu proje, şirket verileri üzerinde Apriori algoritması ve hibrit öneri sistemleri kullanılarak geliştirilen analiz araçlarını içerir.

## Proje İçeriği
- **Apriori Analizi:** Kullanıcı davranışlarından birliktelik kuralları çıkarımı.
- **Veri İşleme:** Büyük ölçekli Parquet ve PKL dosyalarının analize hazırlanması.
- **Raporlama:** Analiz sonuçlarının iş birimlerine sunulacak formatlara dönüştürülmesi.

## Güvenlik Notu
Şirket gizliliği ve veri güvenliği gereği, analizde kullanılan ham veri setleri (CSV, PKL, Parquet) ve özel raporlar bu repoda **paylaşılmamıştır**. Sadece algoritmik yapı ve kod mimarisi yer almaktadır.

## Kurulum
Projenin çalışması için gerekli kütüphaneleri yüklemek için:
```bash
pip install -r requirements.txt
```
